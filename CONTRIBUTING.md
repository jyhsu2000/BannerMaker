# Contributing

Pull requests are welcome.  
For major changes, please open an issue first to discuss what you would like to change.  
For code contributions, see [DEVELOPING.md](DEVELOPING.md) for environment setup and a quick-start guide;
[CLAUDE.md](CLAUDE.md) covers the full architecture and conventions.  
For language packs, you can also use [Crowdin](https://crowdin.com/project/bannermaker) for updating or request for new
languages.
